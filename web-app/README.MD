# Auction Site Web UI.  

## Technology Stack.  
   -AngularJS  
   -Webpack  

## Prerequisites.  
  - Node must be installed to use npm.  

## How to Run.  
  - Step into root directory and issue below commands  
     ```$npm install webpack webpack-dev-server --save-dev```  
     ```$npm install jquery angular --save```  
     ```$npm start```
## How to access.  
  - http://[host:8080]
